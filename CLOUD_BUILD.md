# Сборка ShadowGram в облаке (бесплатно, без Mac и без Apple ID)

Цель: получить `Telegram.ipa` и поставить его на айфон друга. Ничего платить не нужно.

## Часть 1. Собрать IPA на GitHub

1. Зарегистрируйся на github.com (бесплатно).
2. Создай **публичный** репозиторий (это важно — для публичных репозиториев сборка на macOS бесплатна; для приватных macOS-минуты платные).
3. Залей туда содержимое папки `Telegram-iOS-master`. Проще всего — установить **GitHub Desktop** (есть под Windows), создать репозиторий из папки и нажать Publish. Файл `.github/workflows/shadowgram.yml` уже лежит внутри — он и есть скрипт сборки.
4. На странице репозитория → вкладка **Actions** → если попросит включить — включи (Enable).
5. Слева выбери **Build ShadowGram IPA** → кнопка **Run workflow** → Run.
6. Жди (сборка Telegram долгая — десятки минут). Когда позеленеет ✅, открой этот запуск → внизу блок **Artifacts** → скачай **ShadowGram-ipa** (это zip, внутри `Telegram.ipa`).

## Часть 2. Поставить IPA на айфон друга (Windows, бесплатно)

1. На Windows-компе (твоём или друга — том, куда подключишь айфон) поставь **iTunes** (нужен для драйверов) и **Sideloadly** (sideloadly.com, бесплатно).
2. Подключи айфон друга кабелем. На айфоне: «Доверять этому компьютеру».
3. Открой Sideloadly, перетащи в него `Telegram.ipa`.
4. В поле Apple ID введи **бесплатный** Apple ID друга (или заведи новый на appleid.apple.com — платить не надо).
5. Нажми **Start**. Sideloadly сам переподпишет и установит приложение.
6. На айфоне: Настройки → Основные → **VPN и управление устройством** → доверь профилю разработчика (Apple ID). После этого приложение запускается.

Готово — на айфоне появится ShadowGram.

## Честные ограничения бесплатного пути

- **7 дней.** Бесплатная подпись живёт неделю. Потом повтори Часть 2 (пересобирать на GitHub не надо — тот же IPA).
- **Максимум 3 приложения** на один бесплатный Apple ID.
- **Часть функций может барахлить** (пуши, шаринг-расширения) — это из-за ограничений бесплатной подписи, не из-за кода.
- **Сборка на GitHub может упасть**, если на облачной машине нет нужного Xcode 26.2 — тогда в файле `.github/workflows/shadowgram.yml` поменяй строку `runs-on: macos-15` на актуальный образ с Xcode 26.2 (см. github.com/actions/runner-images). Или, если совсем не идёт — арендуй облачный Mac на пару часов (небольшие деньги) и собери там.
